1. Tenha em mente que será necessário instalar em seu computador o NodeJs na versão 20.11.0+ para o back-end.
2. O candidato precisa obrigatoriamente utilizar o React com um empacatador, podendo usar o Vite, Webpack ou similar
3. Deixe o projeto React dentro da pasta Client para manter o desacoplamento das pasta e arquivos entre back=end e front=end
4. Back-end está sendo executado na porta 5000. O candidato tem liberdade total para alterar a porta no qual o servidor vai escutar.
5. O candidato deve fazer um clone do repositório em sua máquina e subir um fork no seu github, realizando commits com mensagens coerentes
6. Este projeto não possui nenhuma conexão com um banco de dados, sendo assim, não haverá a necessidade de instalar nenhum previamente.
